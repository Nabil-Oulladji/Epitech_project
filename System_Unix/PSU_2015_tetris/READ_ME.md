Tetris - Systeme Unix

Utilisation: ./tetris

Mode help: ./tetris --help

Mode Debug; ./tetris -d

-Possibilité de configuré les touches (voir le mode help).

-Possibilité de créé ses propores tetrimino:
Créé un .tetrimino dans le dossier tetriminos.
[size_x] [size_y] [0 < color < 8]
